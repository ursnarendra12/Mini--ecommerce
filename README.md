# E-commerce Service (Spring Boot)

## Overview
This project is a backend service built using Spring Boot that simulates a mini e-commerce system. It handles product management, user operations, and basic API workflows.

## Architecture
- Controller → Service → Repository layered architecture
- RESTful API design
- Maven-based project structure

## Features
- Product CRUD operations
- User-related APIs
- Service-layer business logic
- Database integration (if added)

## Tech Stack
- Java
- Spring Boot
- Maven
- REST APIs

## How to Run
1. Clone the repository
2. Run: `mvn spring-boot:run`
3. Access APIs via Postman or browser

## Notes
This project demonstrates backend service design using Spring Boot and microservice-aligned architecture.
